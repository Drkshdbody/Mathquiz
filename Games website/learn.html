Header Files & Preprocessor Directives

Every C++ program starts with including necessary header files. These provide pre-built functions we can use.

C++ Header Files 
#include <iostream>    // For input/output (cin, cout)
#include <cstdlib>     // For rand(), srand()
#include <ctime>       // For time() - seeding random
#include <string>      // For string manipulation
#include <iomanip>     // For setw(), setprecision()
#include <limits>      // For numeric_limits

using namespace std;

What Each Header Does:

  • iostream - Enables console input (cin) and output (cout)
  • cstdlib - Provides random number functions like rand()
  • ctime - Used to seed random numbers with current time
  • string - Allows us to use text strings
  • iomanip - Helps format output nicely (spacing, decimals)
  • limits - Handles input validation to prevent errors

Why "using namespace std"?

This line lets us write cout instead of std::cout. It saves typing but in large projects, it's better to be explicit with std::.

Global Variables & Struct

We use a struct to group related game data together. This is a fundamental concept in C++ for organizing data.

C++ GameStats Structure 
struct GameStats {
    int totalQuestions;    // How many questions to ask (10)
    int correctAnswers;    // Count of correct answers
    int wrongAnswers;      // Count of wrong answers
    int totalScore;        // Player's total score
    int streak;            // Current correct streak
    int bestStreak;        // Best streak achieved
    string difficulty;     // Current difficulty name
};

Why Use a Struct?

A struct groups related variables together into one package. Instead of having 7 separate variables floating around, we have one organized container called GameStats.

We can create a variable like this:

C++ Creating a GameStats variable 
// Initialize all values at once
GameStats stats = {10, 0, 0, 0, 0, 0, "Medium"};

// Access values using dot notation
stats.totalScore += 20;      // Add 20 to score
stats.correctAnswers++;       // Increment correct count

Utility Functions

These are helper functions that perform common tasks throughout the program. They keep our code clean and reusable.

C++ Clear Screen Function 
void clearScreen() {
    // Use system command to clear the console
    #ifdef _WIN32
        system("cls");      // Windows command
    #else
        system("clear");    // Mac/Linux command
    #endif
}

Cross-Platform Code

The #ifdef preprocessor directive checks if we're on Windows or another system. This lets our program work on any computer!

  • Windows uses cls to clear screen
  • Mac/Linux uses clear to clear screen
C++ Pause Screen Function 
void pauseScreen() {
    cout << "\nPress Enter to continue...";
    cin.ignore(numeric_limits<streamsize>::max(), '\n');
    cin.get();
}

How pauseScreen() Works:

  • cin.ignore() - Clears any leftover characters in the input buffer
  • cin.get() - Waits for user to press Enter
  • This prevents the program from rushing past important messages

Game Logic Functions

These are the brain of the game - they generate random numbers, create math problems, and check answers.

C++ Generate Random Number 
int generateNumber(int maxNum) {
    // Returns a random number from 1 to maxNum
    return (rand() % maxNum) + 1;
}

How Random Numbers Work:

  • rand() - Generates a random number (0 to RAND_MAX)
  • % maxNum - Modulo operator limits range to 0 to (maxNum-1)
  • + 1 - Shifts range to 1 to maxNum

Example: If maxNum = 20, we get random numbers from 1 to 20.

C++ Get Operation Based on Difficulty 
char getOperation(int difficulty) {
    char operations[] = {'+', '-', '*', '/'};
    
    if (difficulty == 1) {
        // Easy: only addition and subtraction
        return operations[rand() % 2];
    }
    // Other levels: all four operations
    return operations[rand() % 4];
}

Array Indexing:

The array operations[] holds 4 characters. We use random index (0-3) to pick one:

  • Index 0 = '+' (addition)
  • Index 1 = '-' (subtraction)
  • Index 2 = '*' (multiplication)
  • Index 3 = '/' (division)
C++ Calculate the Correct Answer 
int calculateAnswer(int num1, int num2, char op) {
    switch(op) {
        case '+': return num1 + num2;
        case '-': return num1 - num2;
        case '*': return num1 * num2;
        case '/': return (num2 != 0) ? num1 / num2 : 0;
        default: return 0;
    }
}

Switch Statement:

The switch statement is like a multi-choice selector. It checks the value of op and runs the matching case:

  • Much cleaner than multiple if-else statements
  • The ternary operator ? : prevents division by zero

Main Game Loop

This is where the actual gameplay happens - asking questions, getting answers, and tracking scores.

C++ The playGame() Function 
void playGame() {
    displayDifficultyMenu();
    
    int diffChoice;
    cin >> diffChoice;
    
    // Set parameters based on difficulty
    int maxNum, pointsPerAnswer;
    string difficultyName;
    
    switch(diffChoice) {
        case 1:
            maxNum = 20;
            pointsPerAnswer = 10;
            difficultyName = "Easy";
            break;
        case 2:
            maxNum = 50;
            pointsPerAnswer = 20;
            difficultyName = "Medium";
            break;
        case 3:
            maxNum = 100;
            pointsPerAnswer = 30;
            difficultyName = "Hard";
            break;
        default:
            maxNum = 200;
            pointsPerAnswer = 50;
            difficultyName = "Challenge";
    }
    
    // Initialize game stats
    GameStats stats = {10, 0, 0, 0, 0, 0, difficultyName};
    
    // Game loop - ask 10 questions
    for (int q = 1; q <= stats.totalQuestions; q++) {
        // Generate random numbers and operation
        int num1 = generateNumber(maxNum);
        int num2 = generateNumber(maxNum);
        char op = getOperation(diffChoice);
        
        // Calculate correct answer
        int correctAnswer = calculateAnswer(num1, num2, op);
        
        // Display question and get user answer
        cout << "\nQuestion " << q << ": ";
        cout << num1 << " " << op << " " << num2 << " = ? ";
        
        int userAnswer;
        cin >> userAnswer;
        
        // Check if correct
        if (userAnswer == correctAnswer) {
            stats.correctAnswers++;
            stats.streak++;
            stats.totalScore += pointsPerAnswer;
            
            // Streak bonus every 3 correct
            if (stats.streak % 3 == 0) {
                stats.totalScore += 5;
            }
            
            if (stats.streak > stats.bestStreak) {
                stats.bestStreak = stats.streak;
            }
            
            cout << "‚úÖ CORRECT! +" << pointsPerAnswer << " points\n";
        } else {
            stats.wrongAnswers++;
            stats.streak = 0;  // Reset streak
            cout << "‚ùå Wrong! Answer was: " << correctAnswer << "\n";
        }
    }
    
    // Display final results
    displayGameOver(stats);
}

Key Concepts in the Game Loop:

  • for loop - Repeats 10 times (one for each question)
  • Variable initialization - GameStats struct holds all game data
  • Conditional logic (if/else) - Checks if answer is correct
  • Increment operators (++) - Adds 1 to counters
  • Modulo (%) - Checks if streak is divisible by 3 for bonus

Streak Bonus Logic

stats.streak % 3 == 0 checks if the streak is a multiple of 3 (3, 6, 9, etc). When true, the player gets a +5 bonus!

Main Function - Entry Point

Every C++ program starts executing from the main() function. This is where everything begins!

C++ The main() Function 
int main() {
    // Seed random number generator with current time
    srand(static_cast<unsigned int>(time(0)));
    
    // Display welcome screen
    displayWelcome();
    pauseScreen();
    
    // Main menu loop
    bool running = true;
    
    while (running) {
        displayMainMenu();
        
        int choice;
        cin >> choice;
        
        // Handle invalid input
        if (cin.fail()) {
            cin.clear();
            cin.ignore(10000, '\n');
            choice = 0;
        }
        
        switch(choice) {
            case 1:
                playGame();
                break;
            case 2:
                displayHowToPlay();
                break;
            case 3:
                displayAbout();
                break;
            case 4:
                running = false;  // Exit the loop
                cout << "\nThanks for playing! Goodbye! üëã\n";
                break;
            default:
                cout << "\n‚ùå Invalid choice! Try again.\n";
                pauseScreen();
        }
    }
    
    return 0;  // Program ended successfully
}

Breaking Down main():

  1. srand(time(0)) - Seeds the random generator so we get different numbers each run
  2. while(running) - Keeps showing the menu until user exits
  3. cin.fail() - Checks if user entered invalid input (like letters instead of numbers)
  4. return 0 - Tells the operating system the program ended successfully

Why return 0?

Returning 0 from main() tells the operating system "everything went well." Non-zero values indicate errors. This is a standard convention in C/C++.
